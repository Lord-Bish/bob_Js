font types

Name Description Usage
Direct Fonts This involves adding font directly using "font-family" property. e.g h1 {
font-family: Arial;
}
Web Fonts Using "<link>" This involvees using external fonts online using the <link> tag. e.g <head>
<link href="font url..">
</head>
Web Font Using "@font-face" This involves using a downloaded font in a css ruleset. e.g @font-face {
font-family: your chosing name..;
src: url("downloaded font path..")
format("file format..");
}
